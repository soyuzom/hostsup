## HOSTS 远程下载更新工具 ##

基本功能：

    1、自动远程下载HOSTS源；2、自动更新替换到系统HOSTS；

    3、支持自定义远程HOSTS链接；4、支持去广告AD源；

    4、支持添加自定义HOSTS更新不丢失；5、支持初始化认HOSTS；

使用方法:

    只需下载Hostsup.bat批处理文件到本地执行，即可！
